🚌 Bus Reservation System

📌 Description
The Bus Reservation System is a java full-stack web application that streamlines the process of booking and managing bus tickets. Users can search for buses, reserve seats, manage bookings, and view travel details, while administrators can manage buses, routes, schedules, and reservations through a dedicated dashboard.

✨ Features
- User registration and login
- Secure authentication and authorization
- Search buses by source, destination, and travel date
- View available seats and bus schedules
- Online seat booking
- Cancel bookings
- View booking history
- Admin dashboard for managing buses, routes, schedules, and bookings
- Responsive and user-friendly interface

🚀 Future Enhancements
- Online payment gateway integration
- Email and SMS ticket confirmation
- QR code-based ticket verification
- Real-time seat availability
- Live bus tracking
- Mobile application support
